3. API Connections & Authentication

How your agent connects to third-party services. Every connection uses your own account.

Service Auth Method Scope / Permissions Key Storage

Shopify API Token Custom app with scoped permissions — typically read-only for products, orders, inventory. Write access only where needed (e.g. updating product descriptions). Local .env file

Meta (Facebook/Instagram) OAuth Token Marketing API — ad performance data, campaign management. Scoped per-account. Local .env file

Lightspeed POS API Token Sales data, inventory, product catalog. Read-only unless write access specifically requested. Local .env file

API Tokens (Most Services)

You create the API key in your own account dashboard. You control the permissions. You can revoke it at any time. The key is stored in a local .env file on your Mac Mini — it never leaves your machine.

OAuth (Anthropic, Meta, Google)

You sign in directly on the provider's website. The authentication token is stored locally. We never see your password — the OAuth flow happens entirely between your Mac Mini and the provider.

4. Network & Remote Access

How we reach your machine for setup and maintenance.

🔒 Tailscale (WireGuard® VPN)

What it is: Tailscale creates a private, encrypted network between your Mac Mini and our management machines. It uses WireGuard® — the same protocol trusted by enterprises and governments worldwide.

What it isn't: Not a traditional VPN that routes all traffic. Only direct machine-to-machine connections are possible, and only between explicitly authorized devices on the network.

Technical Details

Protocol

WireGuard® (UDP, Curve25519, ChaCha20-Poly1305)

Open Ports Required

None. All connections are outbound. No firewall changes needed.

Access Method

SSH over Tailscale (port 22, key-based auth preferred)

Network Impact

Minimal. Only management traffic traverses the tunnel. Your regular network traffic is unaffected.

What This Means for Your Network

  • ✓ No open inbound ports on your firewall
  • ✓ No VPN appliance or gateway hardware needed
  • ✓ Mac Mini only needs outbound internet access (HTTPS + UDP)
  • ✓ Doesn't route any of your other devices' traffic
  • ✓ Can be removed instantly by uninstalling Tailscale

5. Access Model — What We Can See

Full transparency on what SolveWorks can and cannot access.

✅ What We Can Access

  • →
    SSH to the Mac Mini — for setup, updates, troubleshooting, and maintenance
  • →
    Agent logs — to debug issues and monitor health
  • →
    Agent config files — the settings that control how the agent behaves
  • →
    Dashboard data files — the aggregated metrics your dashboard displays

🚫 What We Cannot Access

  • →
    Your other devices — Tailscale only connects to the Mac Mini, not your network
  • →
    Your third-party accounts directly — we don't have login credentials to Shopify, Klaviyo, etc.
  • →
    Other machines on your network — the Tailscale tunnel is point-to-point only
  • →
    Your network traffic — we can't see, log, or monitor anything else on your network

⚡ Regarding API keys on the Mac Mini: While we have SSH access to the machine (and could theoretically read the local .env file), API keys are stored there for the agent to use — not for us. You can rotate or revoke any API key at any time through your own service dashboards. We encourage regular key rotation as a best practice.

6. Messaging & Communication

How you interact with your agent daily.

Telegram Bot

Your AI agent communicates through a Telegram bot that you own. You create it, you control it, and it only responds to whitelisted Telegram user IDs.

The agent does not respond to anyone who isn't on the allowlist — even if someone finds the bot username. Unauthorized messages are silently ignored.

Transport

HTTPS (TLS 1.2+) to Telegram API servers

Authentication

User ID allowlist — only approved users can interact

Message Storage

Conversation history stored locally on Mac Mini only

7. AI Provider (Anthropic)

How the AI model processes your data.

SolveWorks agents use Anthropic's Claude as the AI model. Your agent sends prompts to Anthropic's API and receives responses — similar to how any app uses an API.

What Anthropic receives

  • → The current conversation context (your message + relevant agent memory)
  • → Tool calls (e.g. "search the web for X" or "read this file")

What Anthropic does NOT do

  • ✓ Does not train models on API inputs
  • ✓ Does not store conversations beyond 30 days (for abuse monitoring)
  • ✓ Does not share data with third parties

📄 Anthropic's full privacy policy: anthropic.com/policies/privacy
Commercial terms (API): anthropic.com/policies/terms

Your own Anthropic account: You subscribe directly to Anthropic's Max plan (~$100 USD/mo). You have full visibility into usage and billing through your Anthropic dashboard. SolveWorks does not mark up or resell AI access.

8. Offboarding & Data Removal

What happens if you stop using SolveWorks.

Because everything runs on your hardware:

  • 1. Uninstall Tailscale — instantly revokes our remote access. One click.
  • 2. Revoke API keys — rotate or delete keys in each service dashboard (Shopify, Klaviyo, etc.)
  • 3. Cancel Anthropic subscription — through your own account at console.anthropic.com
  • 4. Keep or wipe the Mac Mini — your data is on it. Keep it as a computer, or factory reset it. Your choice.

There is nothing to "request deletion" from SolveWorks — we don't hold your data. The hosted dashboard (GitHub Pages) can be removed on request or you can delete the repository yourself.

9. Frequently Asked Questions

Does SolveWorks store any of my data on your servers? +

No. We do not operate servers that store client data. All data — API keys, business data, conversation history, agent memory — lives on your Mac Mini. The only externally hosted component is the dashboard, which displays aggregated metrics (not raw data) on a password-protected GitHub Pages site.

Can you see my Shopify orders / Klaviyo emails / customer data? +

We have SSH access to the Mac Mini for maintenance, which means we technically could view files on the machine. However, we do not routinely access client business data. API keys on the machine are there for your agent to use. You can revoke any API key at any time through your own service dashboard, and we encourage regular rotation.

What ports need to be open on our firewall? +

None. The Mac Mini only makes outbound connections. Tailscale uses outbound UDP (port 41641 by default, falls back to HTTPS/443 if UDP is blocked). All API connections are outbound HTTPS (443). No inbound firewall rules are required.

Can the AI agent access other machines on our network? +

No. The Tailscale connection is point-to-point — it only connects the Mac Mini to the SolveWorks management network. It does not bridge your local network. The Mac Mini operates as a standalone device. If you want to further isolate it, you can place it on a separate VLAN — though this isn't required.

How do I revoke SolveWorks' access? +

Uninstall Tailscale from the Mac Mini. This immediately and permanently revokes our remote access. There is no "deactivation request" needed — it's a local uninstall that takes effect instantly. You can also change the Mac's login password as an additional measure.

Is the AI trained on my data? +

No. Anthropic's commercial API terms explicitly state that API inputs are not used to train their models. Your conversations and business data are processed to generate responses and then retained for up to 30 days solely for abuse monitoring, per Anthropic's standard policy.

Can we place the Mac Mini on a separate VLAN? +

Absolutely. The Mac Mini only needs outbound internet access (HTTPS and UDP). It doesn't need to communicate with any other device on your local network. A dedicated VLAN or guest network works perfectly.
