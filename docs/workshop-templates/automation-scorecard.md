## Scoring Methodology

Each opportunity is scored across 4 dimensions on a 1-5 scale. Higher = better candidate for automation.

| Dimension | 1 (Low) | 3 (Medium) | 5 (High) |
|-----------|---------|------------|----------|
| **ROI Impact** | Saves < 1 hr/week | Saves 3-5 hrs/week | Saves 10+ hrs/week or generates revenue |
| **Feasibility** | Requires custom dev, new tools | Some integration work | Off-the-shelf or existing tools |
| **Simplicity** | Highly variable, judgment-heavy | Some rules + some judgment | Rule-based, repetitive, predictable |
| **Speed to Value** | 3+ months to see results | 1-2 months | < 30 days |

**Priority Score** = (ROI × 2) + Feasibility + Simplicity + Speed to Value  
*Maximum score: 25. ROI is double-weighted because impact matters most.*
