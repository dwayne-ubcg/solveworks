SolveWorks Setup

Your AI assistant, installed in one command. Interactive setup guide below.

Run this in Terminal:

curl -sL https://solveworks.io/install.sh | bash

Or with client config:

curl -sL https://solveworks.io/install.sh | bash -s -- --repo https://github.com/your-org/config.git --client your-name

The script automatically installs Homebrew, Node.js, Git, OpenClaw, creates your workspace, and hardens security. The steps below cover the interactive parts the script can't do for you.

1 Configure OpenClaw Interactive

After the install script finishes, run:

openclaw config

The wizard walks through these screens:

a) Workspace

Accept the default (~/clawd) or press Enter.

b) Model Provider

Select Anthropic. Paste your API key when prompted. Your installer will provide this.

c) Default Model

Choose claude-sonnet-4-20250514 (recommended default).

d) Gateway

Select Enable Gateway. It generates a token ‚Äî save this token, you'll need it for the dashboard.

e) Channels

Select Telegram. It will ask for a bot token ‚Äî you'll create this in Step 2.

You can skip channels for now and add them later by re-running openclaw config.

f) Daemon

Select Enable to run OpenClaw as a background service.

2 Create Telegram Bot

Open Telegram and message @BotFather:

  1. Send /newbot
  2. Name: Your Name - Mika (e.g., "Drew - Mika")
  3. Username: yourname_mika_bot (must end in bot)
  4. BotFather gives you a token like 7123456789:AAH...
  5. Copy that token ‚Äî paste it when openclaw config asks for Telegram bot token
Also send /setprivacy to @BotFather ‚Üí select your bot ‚Üí Disable (so the bot can see group messages).

3 Get Your Chat ID

Send any message to your new bot in Telegram, then run:

curl -s https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates | python3 -m json.tool

Look for "chat":{"id": 123456789} ‚Äî that's your chat ID. Your installer may need this to configure direct messaging.

4 Start the Gateway & Approve Pairing

openclaw gateway start

If your Mac Mini shows as a pending node, approve it:

openclaw gateway approve

5 Connect Dashboard

Go to app.openclaw.com and sign in.

  1. Click Add Gateway
  2. Paste the gateway token from Step 1d
  3. Your Mac Mini should appear as connected

6 Add Cron Jobs via Dashboard

In the dashboard, go to Cron and add scheduled tasks. Common ones your installer will set up:

  • Heartbeat check (every 30 min)
  • Email monitoring
  • Calendar reminders

"command not found" after Homebrew install

Close Terminal completely and reopen it. The PATH update only takes effect in new sessions. If it still doesn't work, run:
eval "$(/opt/homebrew/bin/brew shellenv)"

Rate limiting on new Anthropic accounts

This is normal for new API accounts. Limits increase automatically within 24‚Äì48 hours as you add credits and build usage history. No action needed.

"openclaw init" doesn't exist

Skip it ‚Äî that command doesn't exist. The workspace is created by the install script. Use openclaw config for configuration.

"openclaw auth google" doesn't exist

Skip it ‚Äî Google auth is configured separately through the dashboard or browser. Your installer will handle this later if needed.

Tailscale won't install from App Store

Download directly from tailscale.com/download/mac instead. The standalone installer works better than the App Store version.

Gateway "pairing required" error in CLI

Use the dashboard instead. Some CLI operations require an active gateway pairing ‚Äî the dashboard handles this automatically.
