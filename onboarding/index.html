Your Systems

Connect Your Tools

Your AI connects via API keys — no passwords needed. Select what you use and we'll show you how to get the key.

Your keys stay on your hardware. API keys are stored locally on your dedicated Mac Mini — they never leave your machine. You can revoke any key at any time.

⚡
Lightspeed API Key
📋 How to get your Lightspeed token
  1. Log in to your Lightspeed Retail back office
  2. Go to Setup → Personal Tokens
  3. Click Add Personal Token
  4. Name it "SolveWorks" → click Generate
  5. Copy the token — you'll only see it once
🛍️
Shopify POS API Key
📋 How to get your Shopify token
  1. Go to your Shopify Admin → Settings → Apps and sales channels
  2. Click Develop apps → Create an app named "SolveWorks"
  3. Click Configure Admin API scopes → select access needed
  4. Click Install app → copy the Admin API access token
⬜
Square API Key
📋 How to get your Square token
  1. Go to developer.squareup.com
  2. Sign in → go to your Application
  3. Under Credentials, switch to Production
  4. Copy the Access Token
🛍️
Shopify Store API Key
📋 How to get your Shopify token
  1. Go to your Shopify Admin → Settings → Apps and sales channels
  2. Click Develop apps → Create an app named "SolveWorks"
  3. Configure Admin API scopes → Install app
  4. Copy the Admin API access token
◼
Squarespace API Key
📋 How to get your Squarespace token
  1. Go to Settings → Advanced → Developer API Keys
  2. Click Generate Key
  3. Name it "SolveWorks" → set Read permissions
  4. Copy the key
🔮
WooCommerce API Keys
📋 How to get your WooCommerce keys
  1. Go to WooCommerce → Settings → Advanced → REST API
  2. Click Add key, description: "SolveWorks", permissions: Read
  3. Click Generate API key
  4. Copy both the Consumer key and Consumer secret
📧
Omnisend API Key
📋 How to get your Omnisend key
  1. Log in to Omnisend
  2. Go to Store Settings → Integrations & API
  3. Scroll to API Keys → click Create API Key
  4. Name it "SolveWorks" → copy the key
📧
Klaviyo Private API Key
📋 How to get your Klaviyo key
  1. Log in to Klaviyo
  2. Click your account name → Settings → API Keys
  3. Click Create Private API Key
  4. Name it "SolveWorks" → set Full Access → Create
  5. Copy the key
🐵
Mailchimp API Key
📋 How to get your Mailchimp key
  1. Log in to Mailchimp
  2. Click your profile icon → Account & billing
  3. Go to Extras → API keys
  4. Click Create A Key → name it "SolveWorks"
  5. Copy the key
