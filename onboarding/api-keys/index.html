Section 2

Connect Your Tools

Select the tools you use, then provide API keys so your agent can connect securely. Only select what applies to you.

Your keys stay on your hardware. API keys are stored locally on your dedicated Mac Mini — they never leave your machine or touch our servers. You can revoke any key at any time from the provider's dashboard.

🛍️
Shopify
Admin API access token
📋 How to get your Shopify API key
  1. Go to your Shopify Admin → Settings → Apps and sales channels
  2. Click Develop apps (top right) → Create an app
  3. Name it "SolveWorks"
  4. Click Configure Admin API scopes → select the access your agent needs (we'll help with this)
  5. Click Install app → copy the Admin API access token
⚡
Lightspeed
Personal access token
📋 How to get your Lightspeed API key
  1. Log in to your Lightspeed Retail back office
  2. Go to Setup → Personal Tokens
  3. Click Add Personal Token
  4. Name it "SolveWorks" → click Generate
  5. Copy the token — you'll only see it once
📊
Pipedrive
Personal API token
📋 How to get your Pipedrive API key
  1. Log in to Pipedrive
  2. Click your profile icon (top right) → Personal preferences
  3. Go to the API tab
  4. Copy your Personal API token
🟠
HubSpot
Private app access token
📋 How to get your HubSpot API key
  1. Log in to HubSpot
  2. Go to Settings → Integrations → Private Apps
  3. Click Create a private app
  4. Name it "SolveWorks"
  5. Under Scopes, select the access needed (we'll guide you)
  6. Click Create app → copy the access token
🚀
Apollo.io
API key for prospecting and sequences
📋 How to get your Apollo API key
  1. Log in to Apollo.io
  2. Click Settings (gear icon) → Integrations
  3. Go to API Keys
  4. Click Create New Key → name it "SolveWorks"
  5. Copy the API key
💼
LinkedIn Sales Navigator
Session cookie for automation
📋 How to provide LinkedIn access
  1. LinkedIn doesn't offer public API keys for Sales Navigator
  2. We'll set this up together during your onboarding call
  3. For now, just confirm you have an active Sales Navigator subscription
📧
Omnisend
API key for email marketing
📋 How to get your Omnisend API key
  1. Log in to Omnisend
  2. Go to Store Settings → Integrations & API
  3. Scroll to API Keys
  4. Click Create API Key → name it "SolveWorks"
  5. Copy the key
📧
Klaviyo
Private API key
📋 How to get your Klaviyo API key
  1. Log in to Klaviyo
  2. Click your account name (bottom left) → Settings
  3. Go to API Keys
  4. Click Create Private API Key
  5. Name it "SolveWorks" → set Full Access → Create
  6. Copy the private key
🐵
Mailchimp
API key
📋 How to get your Mailchimp API key
  1. Log in to Mailchimp
  2. Click your profile icon → Account & billing
  3. Go to Extras → API keys
  4. Click Create A Key
  5. Name it "SolveWorks" → copy the key
📗
QuickBooks
OAuth connection
📋 How QuickBooks connects
  1. QuickBooks uses OAuth — no API key needed from you
  2. We'll send you a secure authorization link during setup
  3. Click the link, log in to QuickBooks, and approve access
  4. For now, just confirm you use QuickBooks below
📞
RingCentral
API credentials for call data
📋 How to get your RingCentral API key
  1. Go to developers.ringcentral.com
  2. Sign in with your RingCentral account
  3. Click Console → Create App
  4. Name it "SolveWorks", select REST API App
  5. Copy the Client ID and Client Secret
